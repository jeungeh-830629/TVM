🎬 TVM Pro (Thumbnail Video Manager Pro)
TVM Pro는 대량의 동영상 파일을 효율적으로 관리하고, 고해상도 3x3 그리드 썸네일을 통해 직관적으로 파일을 정리할 수 있는 Windows용 통합 솔루션입니다.
✨ 주요 기능
1. 고해상도 썸네일 엔진
1980x1980 그리드: 3x3 형태의 초고화질 썸네일을 생성하여 영상의 내용을 한눈에 파악할 수 있습니다.
스마트 동기화 (Sync): 이미 생성된 썸네일은 건너뛰고, 새 파일만 작업하며, 삭제된 영상의 썸네일은 자동으로 정리합니다.
2. 강력한 파일 관리
윈도우 트리 탐색: 익숙한 폴더 트리 구조로 디렉토리를 탐색합니다.
다중 정렬 지원: 이름(가나다순), 용량(큰/작은 순), 날짜(최신/오래된 순) 기준으로 즉시 정렬이 가능합니다.
일괄 작업: 여러 파일을 선택하여 한 번에 이동하거나 삭제할 수 있습니다. (드래그, Ctrl/Shift 선택 지원)
3. 성능 및 안정성
멀티코어/GPU 가속: 시스템 코어 수에 맞춰 병렬 처리를 수행하며, 하드웨어 가속(CUDA/OpenCL)을 지원합니다.
SQLite 데이터베이스: 메타데이터를 DB로 관리하여 수천 개의 파일도 끊김 없이 로딩합니다.
백신 오탐지 방어: 안전한 파일 저장 로직 및 폴더형 빌드 구조로 백신 차단 문제를 최소화했습니다.
🛠️ 설치 및 빌드 방법 (개발자용)
1. 사전 준비
Python 3.10+ 버전이 설치되어 있어야 합니다.
Inno Setup (설치 파일 제작용, 선택 사항): 다운로드
2. 가상환경 설정 및 의존성 설치
터미널(PowerShell)에서 다음 명령어를 실행합니다:
# 가상환경 생성
python -m venv venv

# 가상환경 활성화
.\venv\Scripts\Activate.ps1

# 필수 라이브러리 설치
pip install -r requirements.txt


3. 실행 파일(.exe) 빌드
백신 프로그램의 실시간 감시를 잠시 끄고 아래 명령어를 실행하세요:
python build_exe.py

빌드가 완료되면 dist/TVM 폴더 안에 실행 파일이 생성됩니다.

TVM.exe를 실행합니다.
왼쪽 트리에서 동영상이 있는 폴더를 클릭하면 스캔이 시작됩니다.
